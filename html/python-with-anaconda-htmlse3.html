3.1 Downloading Youtube Videos

This Section will show how to download a video from youtube using commands as shown below11 On Windows, use Anaconda Powershell Prompt.:

$ conda create --name youtube-dl
$ conda activate youtube-dl
(youtube-dl) $ conda install python==3.7
(youtube-dl) $ python -m pip install youtube-dl
(youtube-dl) $ cd ~/Downloads
(youtube-dl) $ youtube-dl --list-formats https://www.youtube.com/watch?v=aqz-KE-bpKQ
(youtube-dl) $ youtube-dl -f 18 https://www.youtube.com/watch?v=aqz-KE-bpKQ

Note that most Linux distributions doesn’t shipped with multimedia packages. You may need to install them in order to play the downloaded video:

$ sudo apt install mpv
